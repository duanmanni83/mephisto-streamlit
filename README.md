# Mephisto SED Modeling - Streamlit App

基于 Mephisto 的 SED (Spectral Energy Distribution) 建模可视化界面。

## 功能

- 📊 光度数据可视化
- 🔧 SED 模型配置（SFH, SSP, Dust Attenuation 等）
- 📈 交互式 SED 光谱图
- 🤖 Agent 决策历史展示

## 在线访问

[![Streamlit App](https://static.streamlit.io/badges/streamlit_badge_black_white.svg)](YOUR_APP_URL_HERE)

## 本地运行

```bash
pip install -r requirements.txt
streamlit run mephisto_app.py
```

Mephisto 是一个基于大语言模型的 SED 拟合 Agent 框架。
